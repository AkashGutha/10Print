# 10Print

the famous 10 print in various languages


# Usage

For CSharp or C#

``` 
csc 10Print.cs && 10Print.exe
```

For Javascript

```
node 10Print.js
```
For typescript, you first need to install node typings

```
npm install --global ts-node
npm install --save-dev @types/node
```
then run
```
ts-node 10Print.ts
```
